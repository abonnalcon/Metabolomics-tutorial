LC-MS Metabolomics Data Processing

The study used in this tutorial has been previously published as an open access article Chaleckis et al. (2016), in PNAS, and the datasets are available on MetaboLights under the ID MTBLS265.

    Summary¶

  • Peaks detection
  • Peaks grouping
  • Peaks alignment
  • Filling missing features
  • Peak visualisation and metabolites annotation

      • The data processing in this tutorial is done using the XCMS package, more information on the XCMS package is available in the paper by Smith et al. (2006), an online version of XCMS is also available: XCMS Online.

This notebook is based on the R language, multiple libraries such as XCMS, mzR, msdata are available via Bioconductor
